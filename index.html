HTML

  • The head element contains information about the webpage
  • The body element representes the visible content shown to the user
  • Elements that are nested inside other elements are called child elements
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets
  • When possible, we should use semantic HTML elements because they make our HTML easier to read

Git

  • Git is used to control versions of code.
  • There is a whole git workflow that includes the famous pull request
  • To access git, we use the command line; however we always start the commands with git
  • We can use the command git checkout to create branches
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • GitHub is the cloud where we store our code. It includes multiple tools to help us colaborate with other team members
